Custom properties

Las custom properties o variables en css permiten guardar un valor para usarlo luego.

Su declaracion consta de dos guiones seguido del nombre seguido de la definicion. Para usarla se pasa la variable por parametro a la funcion var() Ejemplo en la siguiente etiqueta "code": 

.code-props pre{ 
  --primary-color: rgb(10, 10, 10); 
  --font-size: 32px; 
  background-color: var(--primary-color); 
}

El alcance de las properties esta ligada al bloque en cual se define, si se quiere que sea de alcance global, entonces las properties se tienen que definir en la raiz (:root). Veamos un ejemplo.

Este parrafo tiene el color de fondo definido a partir de una property definida en :root, el codigo es mas o menos asi: 

root:{
  --bg-magenta: rgb(175, 0, 166); 
}

.p-props{
  background-color: var(--bg-magenta);
}
