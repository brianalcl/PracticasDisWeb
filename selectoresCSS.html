Selectores en CSS

Selectores basicos

Selectores con etiquetas

Los selectores con etiqueta se invocan poniendo el nombre de la etiqueta de HTML en la hoja de estilos CSS. Ejemplo el titulo principal (etiqueta h1) y el documento html (etiqueta html) estan editados con las respectivas etiquetas, y en codigo CSS quedaria de esta forma:

html{
background-color: #1e1e1e;
color: rgb(200, 200, 200);
font-family: sans-serif;
scroll-behavior: smooth;
}

h1 {
text-transform: uppercase;
}

Selectores con identificadores (IDs)

Los selectores de identificadores o Ids permiten aplicar estilos mediante el atributo "id" de las etiquetas HTML y son invocadas en CSS con #la-id. Ejemplo este parrafo esta editado con la ayuda de una ID.

Selectores con clases

Los selectores con clases permiten aplicar estilos mediante clases, para poder hacer esto tenemos que definir una clase en la o las etiquetas HTML que queremos modificar, desde CSS se invocan con .la-clase. Ejemplo los siguientes parrafos estan editados mediante clases.

Selectores avanzados

Selectores de hijos

Selectores de hijos directos

Para dar estilos a hijos directos de una etiqueta, clase o id, vasta con poner en CSS el identificador (dependiendo del tipo puede ser la etiqueta la id o la clase) seguido de “>” seguido de la etiqueta a la cual queremos editar. Nota: solo aplicara estilos a los hijos del primer nivel. Veamos un ejemplo con esta lista:

  • Primavera
  • Verano
  • Otoño
  • Invierno

Selectores de hijos descendientes

Para dar estilos a hijos descendientes de una etiqueta, clase o id, vasta con poner en CSS el identificador (dependiendo del tipo puede ser la etiqueta la id o la clase) seguido de “espacio” seguido de la etiqueta a la cual queremos editar. Nota: Aplica a cualquier descendiente. Veamos un ejemplo con esta lista:

  • Primavera
  • Verano
  • Otoño
  • Invierno

Selectores de hermanos

Selector de hermanos en general

Para dar estilos a hermanos (mimo nivel) en cascada de una etiqueta, clase o id, vasta con poner en CSS el identificador (dependiendo del tipo puede ser la etiqueta la id o la clase) seguido de “~” seguido de la etiqueta a la cual queremos editar. Nota: Aplica solo a hermanos descendiente. Veamos un ejemplo con esta lista:

  • Primavera
  • Verano
  • Otoño
  • Invierno

Selector de hermanos adyacentes

Para dar estilos a hermanos adyacentes (mismo nivel, hermano directo) en cascada de una etiqueta, clase o id, vasta con poner en CSS el identificador (dependiendo del tipo puede ser la etiqueta la id o la clase) seguido de “+” seguido de la etiqueta a la cual queremos editar. Nota: Aplica solo a hermanos adyacentes. Veamos un ejemplo con esta lista:

  • Primavera
  • Verano
  • Otoño
  • Invierno

Selectores de atributos

Para dar estilos mediante atributos de una etiqueta, vasta con poner en CSS:
[nombre-del-atributo] para aplicar sobre un atributo ej [href]
[nombre-del-atributo=”tipo”] para aplicar sobre el atributo y tipo ej [target="_blank"]
[nombre-del-atributo*=”P”] para aplicar si la palabra P esta contenida en el atributo ej [href*=".co"]
[nombre-del-atributo^=”P”] para aplicar si el atributo empieza por la palabra P ej [href^="https"]
[nombre-del-atributo$=”P”] para aplicar si el atributo termina con la palabra P ej [href$=".tv"]
[nombre-del-atributo~=”P”] para aplicar si el atributo contiene la palabra P en una lista separada por espacios ej [class~="trabajo"]
[nombre-del-atributo|=”P”] para aplicar si el atributo contiene la palabra P en una lista separada por espacios y que despues del texto venga un "-" ej [class|="red"]

Selector universal

El selector universal permite aplicar estilos a todas las etiquetas del documento, en CSS se nota con “*” pero no se recomienda usarla para dar estilos globalmente, en ocasiones puede usarse localmente ej (.selector-universal * {...}). Ej veamos la lista:

  • Primavera
  • Verano
  • Otoño
